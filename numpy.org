
* Numpy Guide

*NumPy* ek Python library hai jo *high-performance numerical computing* ke liye use hoti hai.
Yeh *multi-dimensional arrays (ndarray)* aur *matrix operations* ko efficiently handle karta hai.

- arch me install: ~sudo pacman -S python-numpy~

*Use Cases:*
- Data Analysis
- Scientific Computing
- Machine Learning & AI
- Image Processing & Computer Vision

** Subplots
1. *Multiple related plots* ek hi figure me compare karne ke liye
2. *Different types of plots* ek hi dashboard jaisa layout banane ke liye
3. *Space saving* – alag-alag plots ko ek hi figure me combine

*** Basic Subplots
#+begin_src python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y1 = [1, 4, 9, 16, 25]
y2 = [25, 20, 15, 10, 5]

# Create 1 row, 2 columns subplots
plt.subplot(1, 2, 1)  # row=1, col=2, plot 1
plt.plot(x, y1, color='blue', marker='o')
plt.title("Square Numbers")

plt.subplot(1, 2, 2)  # row=1, col=2, plot 2
plt.plot(x, y2, color='red', marker='x')
plt.title("Decreasing Numbers")

plt.tight_layout()  # avoid overlap
plt.show()
#+end_src
- *plt.subplot(rows, cols, index)* → choose plot position
- *tight_layout()* → ensures titles/labels don’t overlap
- Multiple types of plots ek figure me

Statistical Data Visualization
Statistical plots big datasets ko *summarize aur samajhne* me help karta hai.
In plots se hum data ka *distribution, spread aur important statistics* aasani se dekh sakte hai.

Box Plot
Box plot ek statistical plot hai jo data ka distribution *five-number summary* ke aadhar par dikhata hai.
5 number summary me ye values hoti hai:
+ *Minimum* – data ki sabse smallest value
+ *First Quartile (Q1)* – 25% data isse neeche hota hai
+ *Median (Q2)* – data ka middle value
+ *Third Quartile (Q3)* – 75% data isse neeche hota hai
+ *Maximum* – data ka biggest value

Isse hame pata chalta hai ki *dat kitna spread hai, meadian kaha hai aur outlier hai ya nahi*.

Simple Box Plot Example
#+begin_src python
import matplotlib.pyplot as plt

scores = [55, 60, 72, 90, 66, 75, 80, 85, 92, 70, 68, 78, 88, 95, 62, 77]

plt.boxplot(scores)

plt.title("Box Plot of Exam Scores")
plt.ylabel("Scores")
plt.show()
#+end_src
- *Box* → middle 50% of data (Q1 to Q3)
- *Line inside box* → median
- *Whiskers* → minimum and maximum (excluding outliers)
- *Dots outside whiskers* → outliers


Box Plot with Multiple Datasets

Animations
Animations use karke hum plots ko time ke saath change hote hue dikha sakte hai.
ye useful hai jab data dynamic ho ya kisi process ka progress dikhana ho.

When to use Animations?
+ *Time-based data* dikhane ke liye (like stock price change)
+ *Simulation ya process* dikhane ke liye
+ *Data changes over time* visualize karne ke liye



Line Drawing Animation
#+begin_src python
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

x = [1,2,3,4,5,6,7]
y = [2,3,5,7,11,13,17]

fig, ax = plt.subplots()

def update(frame):
    ax.clear()
    ax.plot(x[:frame], y[:frame], marker='o')
    ax.set_title("Line Drawing Animation")

ani = FuncAnimation(fig, update, frames=len(x)+1, interval=500)

plt.show()
#+end_src


Growing Bar Chart Animation
#+begin_src python
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

values = [3, 5, 7, 4, 6]

fig, ax = plt.subplots()

def update(frame):
    ax.clear()
    ax.bar(['A','B','C','D','E'], [v + frame for v in values], color='green')

plt.show()
#+end_src

Random Scatter Animation
#+begin_src python
import matplotlib.pyplot as plt
import random
from matplotlib.animation import FuncAnimation

fig, ax = plt.subplots()

def update(frame):
    ax.clear()
    x = [random.randint(1,10) for _ in range(10)]
    y = [random.randint(1,10) for _ in range(10)]
    ax.scatter(x, y, color='red')
    ax.set_xlim(0,10)
    ax.set_ylim(0,10)
    ax.set_title("Random Scatter Animation")

ani = FuncAnimation(fig, update, frames=20, interval=500)

plt.show()
#+end_src
