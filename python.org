*** How Variables Stored in Memory
- Python me variables *reference hote hain objects ko*, direct value store nahi karte (C me pointers jaisa).

#+begin_src python
a = 10
id(a)

a = 12
id(a)
#+end_src
+ *10* ek object banta hai memory (heap) me
+ *a* us object ka reference hold karta hai

#+begin_src
Variable        Memory (Heap)
---------       --------------
   a   ───────▶   10
                (object)

   a   ───────▶   12
               (new object)

previous object gets cleared by python garbage collector
#+end_src

Same value reference to same object in memory
#+begin_src python
a = 10
b = 10

id(a) == id(b)

b = 12
id(a) == id(b)
#+end_src

#+begin_src
   a ───▶ 10 ◀─── b
      (same object)

--------------------------------

Step 1:
   a ─────▶ 10

Step 2:
   a ─────▶ 12
            ▲
         new object
#+end_src


** Python builtin Data Structures

+ *List*  Ordered, mutable (changeable), duplicates allowed
+ *Tuple*  Ordered, immutable (unchangeable), duplicates allowed
+ *Set*  Unordered, mutable, unique elements only
+ *Dictionary*  Key-value pairs, unordered, mutable


*** List Operations

- Ordered *✅*    Mutable *✅*    Duplicates *✅*    Indexing *✅*
#+begin_src python
a = [1, 2, 3, 4, 4]
a

a[0]            # Indexing
a[1:3]          # Slicing
a.append(2)     # adds element in end
a.remove(4)     # removes first occurence of 4
#+end_src

*** Tuple Operations

- Ordered *✅*    Mutable *❌*    Duplicates *✅*    Indexing *✅*
#+begin_src python
t = (1, 3, 2, 2)
t

t[0]         # indexing
t.count(2)   # count
t.index(3)   # find index

t[0] = 100   # ❌ TypeError
#+end_src
Use when data *change nahi hona chahiye*

*** Set Operations

- Ordered *❌*    Mutable *✅*    Duplicates *❌*    Indexing *❌*
#+begin_src python
s = {1, 2, 3, 2}
s

s.add(4)
s.remove(2)

# Set operations
a = {1, 2, 3}
b = {3, 4, 5}

a | b    # union
a & b    # intersection
a - b    # difference
#+end_src

*** Dictionary Operations

- Key-Value *✅*    Mutable *✅*    Keys unique *✅*    Fast lookup *✅*
#+begin_src python
d = {"a": 1, "b": 2}
d

d["a"]        # access value
d["c"] = 3    # add/update
d.keys()
d.values()
d.items()
#+end_src

**** Accessing Invalid key safely
#+begin_src python
d["e"]      #  ❌ KeyError

d.get("e")       # None (no error)
d.get("e", 0)    # 0 (default value)
#+end_src


** Python List Slicing

*** 1D List
#+begin_src python
a = [1, 2, 3, 4, 5, 6, 7, 8]

a[0]
a[-1]
a[-2]

a[2:5]
a[:4]
a[3:]
a[:]

a[1:7:2]
a[7:3:-1]
a[-1:3:-1]

a[::-1]
#+end_src

*** 2D List
#+begin_src python
a = [
    [1,2,3],
    [4,5,6],
    [7,8,9]
]

a[0][0]
a[-1][-1]

# Get row
a[0]
a[-1]

# Reverse Rows
a[0][::-1]
[row[::-1] for row in a]

# Get Column
[row[0] for row in a]
[row[1] for row in a]

[row[1:] for row in a[1:]]
#+end_src


** list comprehensions

*** Basic Syntax
#+begin_src
[expression for item in iterable]
#+end_src
Har *item* par *expression* apply hota hai aur new list ban jaati hai.

#+begin_src python
# Normal Loop
squares = []
for x in range(5):
    squares.append(x*x)

squares

# List Compression
[x*x for x in range(5)]
#+end_src

*** Basic Examples
#+begin_src python
# Even numbers
[x for x in range(10) if x % 2 == 0]

# Even → "even", Odd → "odd"
["even" if x % 2 == 0 else "odd" for x in range(5)]

# string example
[c.upper() for c in "hello"]

# Multiple loops
[(x, y) for x in [1, 2] for y in [3, 4]]
#+end_src

*** Nested List Comprehension
#+begin_src python
[[i for i in range(3)] for j in range(3)]
#+end_src

*** Flatten 2D List (Matrix)
#+begin_src python
matrix = [[1, 2, 3],
          [4, 5, 6],
          [7, 8, 9]]

[num for row in matrix for num in row]
#+end_src

*** Transpose of Matrix
#+begin_src python
[[row[i] for row in matrix] for i in range(3)]
#+end_src


** Dictionary Comprehension

*** Basic Syntax
#+begin_src
{key_expression: value_expression for item in iterable}
#+end_src
Har item ke liye ek *key-value pair* generate hota hai.

#+begin_src python
{x: x*x for x in range(5)}
{x: x*x for x in range(10) if x % 2 == 0}
{x: ("even" if x % 2 == 0 else "odd") for x in range(5)}
#+end_src

*** From existing list
#+begin_src python
lst = ["apple", "banana", "cherry"]

{name: len(name) for name in lst}

{i: val for i, val in enumerate(lst)}
#+end_src

*** Swap keys and values
#+begin_src python
d = {"a": 1, "b": 2}

{v: k for k, v in d.items()}
#+end_src

*** Nested Dictionaries
#+begin_src python
{x: {y: x*y for y in range(3)} for x in range(3)}
#+end_src

*** Filtering existing dictionaries
#+begin_src python
d = {"a": 1, "b": 2, "c": 3}

{k: v for k, v in d.items() if v > 1}
#+end_src
